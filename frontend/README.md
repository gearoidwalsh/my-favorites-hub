# Favorites Hub - Frontend

React application for displaying and managing your favorite content.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file (optional, for production):
```
REACT_APP_API_URL=http://localhost:5000
```

3. Start development server:
```bash
npm start
```

4. Build for production:
```bash
npm run build
```

## Features

- Gallery view with filtering
- YouTube video embedding
- Search functionality
- Like system
- Admin panel for content management
- Responsive design

## Environment Variables

- `REACT_APP_API_URL` - Backend API URL (defaults to http://localhost:5000)
